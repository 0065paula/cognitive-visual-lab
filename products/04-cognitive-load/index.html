Why too much information at once makes the brain "freeze" · Printable Plate

I. THE LOADS · 三种负荷 THE LOADS · Three Types
Fig. 1
内在负荷 Intrinsic
40%
材料本身的复杂度。学微积分比学加法负荷高——这是不可避免的。
Complexity inherent to the material. Calculus is harder than addition — unavoidable.
外在负荷 Extraneous
35%
不良设计带来的浪费。花哨的 PPT 动画、无关的插图、分散注意的装饰。
Waste from poor design. Fancy PPT animations, irrelevant images, distracting decoration.
相关负荷 Germane
25%
促进深层理解的投入。建立联系、归纳模式、纳入已有知识体系。
Investment in deep understanding. Building connections, recognizing patterns, integrating knowledge.

教学设计的核心目标：降低外在负荷，把释放出的认知资源用于相关负荷。内在负荷由内容决定，无法消除，但可以通过「分块」和「渐进」来管理。

The core goal of instructional design: Reduce extraneous load, redirect freed resources to germane load. Intrinsic load is determined by content and cannot be eliminated, but can be managed through "chunking" and "scaffolding."

II. OBSERVATIONS · 观察 OBSERVATIONS · Observations
Fig. 2
Obs. 1
总容量固定，分配是关键——工作记忆就像 4±1 个槽位（Cowan, 2001）。内在负荷占得越多，留给相关理解的就越少。外在负荷是「小偷」，悄悄偷走你的理解力。 Total capacity is fixed; allocation is key — Working memory has ~4 slots (Cowan, 2001). The more intrinsic load consumes, the less remains for deep understanding. Extraneous load is a "thief" silently stealing comprehension.
Obs. 2
分心效应（Split-Attention Effect）——当信息需要同时在两个地方处理（如看图又看文字说明），外在负荷翻倍。整合呈现可以降低 40-60% 的认知负荷。 Split-Attention Effect — When information must be processed in two places simultaneously (e.g., diagram + text explanation), extraneous load doubles. Integrated presentation can reduce cognitive load by 40-60%.
Obs. 3
冗余效应（Redundancy Effect）——同样的信息用两种形式呈现（如同时说和写），初学者反而更差。专家可以处理冗余，初学者会被淹没。 Redundancy Effect — Presenting the same information in two forms (e.g., spoken + written simultaneously) harms novices. Experts can handle redundancy; novices are overwhelmed.
Obs. 4
专家悖论——专家因为知识自动化，内在负荷极低，所以可以同时处理多个复杂任务。新手连一个任务都超负荷。这就是为什么「专家觉得简单」不等于「对初学者简单」。 The Expert Paradox — Experts have automated knowledge, so intrinsic load is minimal, allowing multiple complex tasks. Novices are overwhelmed by one task. "Simple for an expert" ≠ "simple for a beginner."
III. METHODS · 实践方法 METHODS · Practical Methods
Fig. 3
1
一次只给一个目标——教学材料中每个元素都应为学习目标服务。如果一个图表、一段文字、一个动画不能回答「这为什么在这里？」，它就是外在负荷，应该删除。 One goal at a time — Every element in instructional material should serve the learning objective. If a diagram, text, or animation cannot answer "Why is this here?" it's extraneous load and should be removed.
2
用「样例」代替「题目」——给新手直接展示解题步骤（worked examples），比让他们自己摸索更省认知资源。当技能提升后，再逐步减少提示（fading）。 Use worked examples instead of problems — Showing novices worked solution steps saves cognitive resources compared to letting them struggle. Fade prompts gradually as skill develops.
3
整合信息，避免分心——把图和文字说明放在同一视觉区域，不要让人在页面上来回扫视。标注直接画在图上，而非放在图注里。 Integrate information, avoid split attention — Place diagrams and explanations in the same visual area. Don't make readers scan back and forth. Label directly on diagrams, not in captions.
4
渐进释放责任——从「我来做，你看」（示范）→「我们一起做」（指导）→「你做，我来看」（独立）。每一步都在匹配学习者的认知容量。 Gradual release of responsibility — From "I do, you watch" (modeling) → "We do together" (guided) → "You do, I watch" (independent). Each step matches the learner's cognitive capacity.
5
消除冗余，但保留多样性——不要同时用文字+语音+图表说同一件事。但可以用一种形式讲概念，另一种形式讲例子（互补而非重复）。 Eliminate redundancy, but preserve variety — Don't present the same thing in text + audio + diagram simultaneously. But use one format for concepts and another for examples (complementary, not repetitive).

* 核心机制：认知负荷理论（Sweller, 1988）指出，学习发生在相关负荷占主导时——教学设计者的任务是最大化相关负荷，最小化外在负荷。

* Core mechanism: Cognitive Load Theory (Sweller, 1988) states learning occurs when germane load dominates — instructional designers must maximize germane and minimize extraneous load.
